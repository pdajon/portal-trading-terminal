# MVP Scope

## In scope

- Chart-centered interface
- Hyperliquid-only market universe
- Trade planning on chart
- Right panel for trade summary and actions
- Bottom panel for positions and order history
- Minimal execution flow after planning state is ready

## Out of scope

- AI trading coach
- Discipline rules and lockouts
- Edge Engine behavior intelligence
- Progression systems
- Social features
- Creator marketplace
- Order book
- Trades feed
- DOM ladder

## Guardrails

- Protect chart space first
- Keep execution secondary and contextual
- Avoid duplicate information blocks
- Prefer simple, swappable integrations over heavyweight early abstractions
