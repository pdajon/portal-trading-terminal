# Portal

Portal is a chart-first trading terminal built on top of Hyperliquid's backend.

## What Portal is

- A trader workspace built around the chart
- A cleaner alternative to bouncing between TradingView and Hyperliquid
- Focused on discretionary traders who plan trades around levels, structure, and risk

## MVP scope

- Chart-first workspace shell
- Hyperliquid-only symbol universe
- On-chart trade planning with entry, stop loss, and take profit
- Contextual execution flow
- Right-side trade summary panel
- Bottom positions and order history panel

## Out of scope right now

- AI coaching
- Discipline engine
- Edge Engine behavior analysis
- Gamification and progression
- Social or creator systems
- Order book, trades feed, and DOM ladder

## Current milestone

MVP foundation scaffold:

- Next.js app skeleton
- Chart-first layout placeholders
- Shared types
- Mock data for MVP UI development
- Product and architecture docs

## Local setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
