# Portal

Portal is a next-generation crypto trading terminal built on top of Hyperliquid.

It focuses on execution, automation, and trader-defined logic rather than just charting.

## Core Features

- Advanced execution engine (market, limit, and zone-based orders)
- Trigger-based entries (touch + candle close conditions)
- Order Zones (multi-order distribution across price ranges)
- Real-time position management with synchronized TP/SL
- Custom trading logic framework (in development)

## Philosophy

Most trading platforms stop at charting.

Portal is built to answer:
> "What happens after I draw the line?"

## Status

Currently in active development.

Core trading systems are functional:
- execution
- order management
- trigger engine

Charting layer upgrade (TradingView integration) in progress.

## Tech Stack (Current)

- Frontend: React / Next.js
- Backend: Node + Python bridge
- Exchange: Hyperliquid
- Data: WebSocket + custom API layer
